# Задание 5

Программа считывает файл, содержащий изображение, затем изображение переводится в другое цветовое пространство, выделяется наиболее информативный канал, создаётся и применяется маска золотых частей платы, находятся контуры на маске, затем контур помечается путем отрисовки и заполнения контура и отрисовки ограничивающего прямоугольника

# Технологии

* Python

# Использование

Для запуска приложения необходимо загрузить все файлы из папки task3 и удостовериться, что все необходимые библиотеки установлены.

# Разработка

*Требования*

Для работы данной программы необходимо развернуть виртуальное окружение из файла requirements.txt
Для установки необходимо выполнить подобную команду:

pip install -r requirements.txt

Ход работы:

Изображение загружается, переводится в цветовое пространство hsv, в котором выделяется канал v, т.к. он оказался самым информативным 

![изображение](https://github.com/DoorToWar/task3-5/assets/117304018/07934f73-aa32-40c1-a692-80ad70e25066)

Создаётся маска для светлых пикселей, применяется к изображению. 
В массив contours записываются контуры на маске, с помощью цикла выводятся все контуры, производится фильтрация контуров (при фильтрации часть золотой подложки не учитывается в дальнейшем)

На изображении выделяются и заполняются контуры, на другом изоражении контуры выделяются прямоугольниками
